Introduction

WLR UI Library is a custom UI library designed for creating styled buttons in web projects. It provides a variety of button sizes, states, and visual effects to enhance user interfaces. With WLR UI Library, developers can easily implement buttons with different styles by applying predefined classes. These include options for varying button sizes such as small, medium, and large, as well as effects like scaling, rotation, shadow, and transformation. The library's core functionality relies on a simple base class for buttons, and additional features are introduced through customizable classes, making it easy to integrate into any project. WLR UI Library is accessible via a CDN, allowing developers to quickly add its functionality to their web applications.

  • Button Effects: The library offers several effects that can be added to buttons, such as scaling, rotating, shadow, and transformation effects, achieved by adding specific classes to buttons.
  • Responsive Design: WLR UI Library is designed to be responsive, with CSS that adapts the layout of the navigation and main content area based on the screen size, making it mobile-friendly.
  • Flexible Button Sizing: Buttons can be adjusted to different sizes (e.g., x-small, small, medium, large) by applying size-specific classes, providing flexibility in UI design.

    Base class

    Making a button


    WLR UI Library has a base .btn class that sets up basic styles such as padding and content alignment. By default, .btn controls have a transparent border and background color, and lack any explicit focus and hover styles.

    <button class="btn"></button>

    Any button must have the class "btn".

    • btn-xsm
    • btn-sm
    • btn-md

    • btn-xlg

    Adding effects to buttons



    You can add a button effect by adding a effect class.

    The effect classes are

    • btn-transform
    • btn-scale
    • btn-shadow
    • btn-rotate
    • underline-border
    • btn-fill-color
    <button class="btn btn-lg btn-scale">Button</button>
